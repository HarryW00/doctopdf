## 1. Mission & Rules

### Mission

Create `.github/workflows/lint.yml` that runs `ruff check`, `mypy`, and `pytest` on Python 3.11 for every push to `main` and every pull request.

### Rules
- Create ONLY `.github/workflows/lint.yml`.
- Do not modify any other file.
- If `.github/workflows/` already contains a workflow, STOP and report.
- Workers are leaf nodes.

## 2. Context

### Input Files
- `references/ci-tooling.md` — exact workflow snippet and the `pip install -e ".[dev]"` install step (the `dev` extra is provided by #4).

### Background
CI must be green from day one (SPEC #7 Req 2). It installs the project with the `dev` extra (which brings pytest, pytest-cov, ruff, mypy) and runs lint, type-check, and the Word-free test suite on a single Python 3.11 runner.

---

## 3. Tasks

### `.github/workflows/lint.yml` — create

Create the file with exactly this content:

```yaml
name: Lint

on:
  push:
    branches: [main]
  pull_request:

jobs:
  lint:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: '3.11'
      - run: pip install -e ".[dev]"
      - run: ruff check doctopdf/
      - run: mypy doctopdf/ --ignore-missing-imports
      - run: pytest tests/
```

### Output
Report: **Files created** (`.github/workflows/lint.yml`), **risks** (or "None").
