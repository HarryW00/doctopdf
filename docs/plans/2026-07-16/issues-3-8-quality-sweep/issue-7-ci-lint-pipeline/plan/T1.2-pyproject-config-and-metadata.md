# Worker Prompt: T1.2-pyproject-config-and-metadata

- **Source task**: Batch 1 / Task 2 — Standardize `pyproject.toml` on Python 3.11 and add ruff/mypy config

---

## 1. Mission & Rules

### Mission

In `pyproject.toml`: bump `requires-python` to `>=3.11`, remove the 3.9/3.10 classifiers, and add `[tool.ruff]` and `[tool.mypy]` configuration.

### Rules
- Modify ONLY `pyproject.toml`.
- Do NOT touch the `[project.optional-dependencies] dev` list (owned by #4).
- Do NOT add or remove any classifier other than the 3.9 and 3.10 Python ones.
- If a target line is absent or differs, report it — do not guess.
- Workers are leaf nodes.

---

## 2. Context

### Input Files
- `pyproject.toml` — currently `requires-python = ">=3.9"` with Python classifiers for 3.9, 3.10, 3.11, 3.12; no `[tool.ruff]`/`[tool.mypy]`.

### Background
Decision CQ2=A standardizes the project on Python 3.11 (SPEC #7 Req 4). ruff and mypy targets must match so the toolchain, CI runner, and declared support all agree. `ignore_missing_imports = true` keeps mypy from failing on missing third-party stubs (SPEC #7 Req 2).

---

## 3. Tasks

### `pyproject.toml` — bump `requires-python`

1. Find:
   ```toml
   requires-python = ">=3.9"
   ```
2. Replace with:
   ```toml
   requires-python = ">=3.11"
   ```

### `pyproject.toml` — drop 3.9 and 3.10 classifiers

1. In the `classifiers = [...]` array, delete these two lines (leave 3.11 and 3.12):
   ```toml
       "Programming Language :: Python :: 3.9",
       "Programming Language :: Python :: 3.10",
   ```

### `pyproject.toml` — append ruff/mypy config

1. At the end of the file, append:
   ```toml

   [tool.ruff]
   line-length = 100
   target-version = "py311"

   [tool.mypy]
   python_version = "3.11"
   ignore_missing_imports = true
   ```

### Output
Report: **Files modified** (`pyproject.toml`), the before/after of the three changes, **risks** (or "None").

---

## 4. Verification

1. Run: `grep -n 'requires-python' pyproject.toml` → shows `">=3.11"`.
2. Run: `grep -c "Python :: 3.9\|Python :: 3.10" pyproject.toml` → `0`.
3. Run: `grep -nE "target-version|python_version" pyproject.toml` → shows `target-version = "py311"` and `python_version = "3.11"`.
